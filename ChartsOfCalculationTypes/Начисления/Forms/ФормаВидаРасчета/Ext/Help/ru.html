Начисление

В программе предусмотрена возможность регистрации сумм по широкому спектру видов начислений, а также регистрации выплат доходов для различных целей.
Большинство видов начислений сосредоточены в списке начислений. Начисления из этого списка предназначены не только для регистрации сумм, но могут использоваться и для регистрации отработанного и неотработанного сотрудниками времени.

Карточка начисления

Наименование и код позволяют различать начисления в списках программы. Кроме того, на закладке Описание можно задать краткое наименование начисления. Оно будет выводиться в различных отчетах по начислениям, где они фигурируют в заголовках колонок. На этой закладке можно также заполнить произвольное описание начисления в справочных целях.

Назначение начисления

Назначение позволяет программе определять, какие настройки являются допустимыми для этого начисления, каким образом и где это начисление может назначаться, рассчитываться и выполняться. Другими словами, по назначению программа «узнает» начисления.
Список возможных назначений начислений является закрытым, расширить его нельзя. Некоторые из назначений доступны только в том случае, если выбрана соответствующая возможность программы. Это такие назначения, использование которых предполагает также создание специфичного для них показателя, вида времени и т.п., которое выполняется при задействовании возможности.
Большая часть назначений являются специализированными, т. е. начисления с таким назначением используются только для выполнения специализированными «межрасчетными» документами программы:

1. Для начисления документом Больничный лист:
- Оплата больничного листа,
- Оплата больничного листа за счет работодателя,
- Оплата больничного листа, несчастный случай на производстве,
- Оплата больничного листа, профзаболевание,
- Болезнь без оплаты,
- Отпуск по беременности и родам без оплаты.
2. Для начисления документом Отпуск:
- Оплата отпуска,
- Компенсация отпуска (также может начисляться документом Увольнение),
- Материальная помощь к отпуску (доступно, если выбрана соответствующая возможность программы).
3. Для начисления документом Простой сотрудников (доступны только при выбранной возможности учета простоев):
- Простой по вине работника,
- Оплата простоя по вине работодателя,
- Оплата простоя по независящим от работодателя причинам.
4. Для начисления документом Прогул, неявка:
- Неявка по невыясненным причинам,
- Прогул.
5. Для начисления другими документами:
- Оплата командировки (документ Командировка),
- Оплата времени сохраняемого среднего заработка (документ Отсутствие с сохранением оплаты),
- Отпуск без оплаты (документ Отпуск без сохранения оплаты),
- Отгул (одноименный документ, а также документ Отпуск).

Другая часть назначений начислений также обладает определенной спецификой:

- Сдельная оплата труда – может выполняться только при окончательном расчете зарплаты, по умолчанию заполняется формула с использованием показателя сдельной выработки. Назначение доступно, если в настройках расчета зарплаты выбрано использование сдельного заработка.
- Доплата за совмещение и Доплата до среднего заработка – могут назначаться только соответствующими специализированными документами. Назначения доступны, если выбраны такие возможности программы.
- Оплата труда в натуральной форме – может назначаться только в качестве планового начисления, которое будет выполняться ежемесячно, но можно выполнить и документом Доход в натуральной форме.

«Универсальными» назначениями начислений, т. е. теми, которые допускают дополнительное управление использованием начислений с таким назначением, являются:

- Повременная оплата труда и надбавки – для описания начислений, составляющих фонд оплаты труда сотрудников (т.е. являющихся «зарплатой»).
- Компенсационные выплаты – для описания компенсаций, т.е. начислений, не связанных с оплатой труда. Такие начисления, например, особым образом учитываются в расчете ИПН и не учитываются при расчете общего среднего заработка.
- Материальная помощь – для описания начислений, являющихся материальной помощью. Материальная помощь, как и компенсационные выплаты, имеет особенности в обложении ИПН, учета в среднем заработке.
- Премия – для описания премий и вознаграждений, которые, согласно законодательству, особым образом учитываются при расчете общего среднего заработка.
- Доход в натуральной форме – начислениями с таким назначением регистрируются суммы, соответствующие доходу, полученному не в денежной форме. Такие начисления регистрируются с целью обложения ИПН и взносами. Сумму, полагающуюся к выплате сотруднику, они не увеличивают.

Для начислений с этими назначениями можно указать способ их выполнения.

Способ выполнения начисления

Для начислений с назначением Повременная оплата труда и надбавки, Компенсационные выплаты, Материальная помощь, Премия, а также Доход в натуральной форме в поле Начисление выполняется можно указать способ их выполнения:

  • Ежемесячно – такое начисление можно назначить плановым, и оно будет автоматически выполняться при окончательном расчете зарплаты документом Начисление зарплаты и взносов и др. Можно отметить показатели для сотрудника постоянного использования, значения которых будут запрашиваться при назначении начисления плановым.
  • По отдельному документу – такое начисление можно выполнить только межрасчетными документами Разовое начисление, Материальная помощь, Премия или Доход в натуральной форме (соответственно назначению).
  • В перечисленных месяцах – совпадает со способом Ежемесячно. Отличие заключается в том, что можно выбрать месяцы, при окончательном расчете которых будет выполняться это начисление.
  • Только если введено значение показателя – такое начисление будет выполняться при окончательном расчете только в том случае, если на этот месяц с помощью универсального документа Данные для расчета зарплаты введено значение хотя бы одного из отмеченных «определяющих» показателей. В формуле такого начисления должен использоваться показатель для сотрудника, разового использования.
  • Только если введен вид учета времени – такое начисление будет выполняться при окончательном расчете только в том случае, если на этот месяц каким-либо способом (в графике, табеле или в документе Данные для расчета зарплаты) зарегистрировано время выбранного вида.
  • Только если время приходится на праздничные дни – такое начисление будет выполняться при окончательном расчете только в том случае, если рабочее время по графику работы сотрудника (обычному или индивидуальному) выпадает на государственные праздники (согласно производственному календарю).

Таким образом, другим важным параметром начисления, который может определять не только то, как будет рассчитываться его результат, но и как будет выполняться начисление, является его формула.

Формула расчета и показатели

Для начислений с некоторыми назначениями алгоритм расчета регламентирован законодательством и заложен внутри программы, поэтому для них нельзя описать произвольную формулу расчета. Например, Оплата больничного листа.
Для других видов назначений допустимо описать способ их расчета произвольной формулой. При этом для «специализированных» назначений формула расчета предлагается программой автоматически, но при необходимости ее можно изменить.
При создании начислений с «универсальным» назначением Повременная оплата труда и надбавки и др. можно описать формулу их расчета самостоятельно. При этом формула не только будет определять алгоритм их расчета, но и используемые в формуле показатели могут также выступать в качестве дополнительного условия для способа их выполнения.

Если результат начисления не рассчитывается, а вводится в программу готовой суммой (т.е. фиксированной или рассчитанной вне программы), то нет необходимости описывать его формулу, которая фактически должна состоять из одного показателя. Достаточно указать, что результат вводится фиксированной суммой, и тогда в документах можно будет сразу ввести результат.

Если начисление можно описать формулой, следует выбрать вариант Результат рассчитывается. В этом случае следует описать формулу начисления в отдельной форме, которая вызывается по ссылке Редактировать формулу.

Для начислений, которые выполняются Ежемесячно или В перечисленных месяцах, среди используемых в формуле можно отметить показатели для сотрудника постоянного использования, значения которых будут запрашиваться при назначении начисления плановым.
Для начислений, которые выполняются только если введено значение показателя следует отметить «определяющий» показатель. Показатель выбирается среди показателей разового использования для сотрудника, который обязательно должен фигурировать в формуле такого начисления.

Прочие возможности

Для начислений, расчет которых возможен не только в целом за месяц при окончательном расчете, доступен флажок Начисляется при расчете первой половины месяца. Если он установлен, начисление будет учитываться при расчете суммы аванса документом Начисление за первую половину месяца.

Для начислений, которые могут быть назначены в плановом порядке, доступен флажок Поддерживает несколько одновременно действующих начислений. Его следует установить, если это начисление требуется назначать одному и тому же сотруднику параллельно с разными периодами или размерами. Такие начисления следует назначать документом Назначение планового начисления.

Для начислений, которые могут быть назначены в плановом порядке, также доступен флажок Включать в ФОТ. Если флажок не установлен, то начисление не будет учитываться при формировании планового ФОТ сотрудников.

Если начисление предполагает расчет общего среднего заработка, то в разделе Расчетный период среднего заработка можно установить длительность расчетного периода, отличную от установленной законодательством.

Если в формуле начисления используются показатели, которые накапливаются по отдельным значениям документами ввода данных для расчета зарплаты или поставляемый показатель СдельныйЗаработок, то становится доступна настройка Значения накапливаемых показателей подбираются:
- по соответствию даты регистрации интервалу начисления – при расчете начисления будут заполнены значения таких показателей, пришедшиеся ровно на его период (который может и не совпадать с целым месяцем);
- все накопленные значения за месяц – начисление будет относиться ко всему месяцу в целом и не сможет иметь отличный от всего месяца период. Значения показателей будут «собраны» за весь месяц в целом, независимо от того, какой период сотрудник фактически работал.

Например, при настройке начисления Сдельный заработок:
- если планируется использовать «сводную» регистрацию сдельной выработки, т.е. вводить наряды не по дням, а целиком за некоторый период, то следует установить настройку начисления в положение все накопленные значения за месяц;
- если планируется регистрировать сдельную выработку «точно», т.е. по дням, то настройку можно установить в по соответствию даты регистрации интервалу начисления. При этом следует учитывать, что если по какой-то причине выработка будет зарегистрирована в день, когда сотрудник фактически отсутствовал, то при расчете зарплаты она учтена не будет.

Если в периоде вводятся межрасчетные документы, рассчитываемые методом "от обратного" с установленным признаком "Учитывать прямые начисления" и межрасчетные документы с прямыми начислениями по одному и тому же сотруднику, то необходимо в документах прямых начислений указывать способ выплаты "В межрасчетный период".

Например, если в одном месяце по одному физическому лицу рассчитываются Больничный (прямой метод расчета) и Премия методом расчета "от обратного" (оба документа со способом выплаты "В межрасчетный период"), то в документе Больничный лист необходимо указать способ выплаты "В межрасчетный период", чтобы отразить применение вычетов прямых начислений в регистрах налогов.
Далее зарегистрировать документ Премия, в котором, благодаря отражению данных по вычетам в больничном, будут корректно рассчитаны сумма "от обратного" и межрасчетные налоги/взносы.
В описанной ситуации вычет по ИПН в размере 1 МЗП при расчете премии не будет применен или будет применен частично, на остаток суммы вычета после расчета больничного и премии, как в расчетах от обратного, так и в расчете самих межрасчетных налогов/взносов.

В противном случае, сумма ИПН в начислении "от обратного" в документе Премия будет рассчитана корректно (с учетом данных примененного вычета по сумме больничного), но сумма самого налога ИПН будет рассчитана не корректно, т.к. в налоговых регистрах информация о примененном вычете ИПН в документе Больничный лист будет отсутствовать.

Закладка "Расчет базы"

Закладка доступна, если при расчете начисления предполагается использование результатов расчета других начислений, т.е. в его формуле используется показатель РасчетнаяБаза.
На закладке перечисляется как список начислений, результаты расчета которых необходимо учесть при расчете текущего начисления, так и период, за который следует брать эти результаты.
Если начисление выполняется по отдельному документу, то текущий месяц (и периоды, включающие текущий месяц) для выбора недоступны. В противном случае имелся бы риск при расчете этого начисления не учесть результаты расчета базовых начислений, которые еще не были зарегистрированы в текущем месяце. Таким образом, при расчете начислений от базы в межрасчетный период предполагается использование результатов только уже прошедших периодов.

Закладка "Зависимости"

Начисление может не только использовать при своем расчете результаты расчета других начислений, но и также использоваться при расчете других начислений и удержаний. Для удобства просмотра и редактирования начислений и удержаний, в расчетную базу которых входит текущее начисление, предусмотрены списки зависимых видов расчета. При добавлении вида расчета в этот список текущее начисление попадает в список его базовых (на закладку Расчет базы этого вида расчета).

Закладка "Приоритет"

В программе предполагается ведение учета отработанного сотрудником времени методом отклонений. Метод отклонений подразумевает назначение сотруднику начисления, которое полагается ему за полностью отработанное по плановому графику работы время (например, оклад). Отклонения от графика по причине невыходов регистрируются соответствующими документами (отпуск, больничный и т.п.).
На этой закладке указывается какие начисления должны выполняться вместо текущего или начисления, вместо которых выполняется текущее начисление.
Как правило, эти таблицы заполняются программой автоматически по результатам анализа основных параметров начислений.

Закладка "Средний заработок"

В разделе Расчет оплаты отпусков, командировок и др. для просмотра или изменения доступны настройки, влияющие на расчет среднего заработка для этих случаев.
Подробнее о расчете общего среднего заработка и влиянии этих настроек см. в разделе Расчет среднего заработка для отпусков, командировок и др.

При расчете квартальной премии как периодического начисления с использованием документа Начисление зарплаты и взносов на закладке Средний заработок необходимо выбрать один из следующих вариантов периода:
- Текущий квартал,
- Предыдущий квартал,
- Несколько предыдущих месяцев.
Использование других вариантов может привести к тому, что сумма квартальной премии не будет учтена при расчете среднего заработка. Период выплаты премии в документах окончательного расчета определяется относительно месяца, в котором отражено начисление премии.

Закладка "Налоги, взносы, бухучет"

В разделе Налоги и взносы указывается:
- Вид дохода. Указывается вид дохода, облагаемого у источника выплаты, используется для определения размера ставки ИПН.
- Способы обложения. Содержат указание на то, является или нет начисление объектом обложения индивидуальным подоходным и социальным налогом, удерживаются или нет с него обязательные пенсионные взносы, обязательные профессиональные пенсионные взносы и социальные отчисления.

Ограничение

Отсутствует возможность расчета методом "от обратного" для видов расчетов:

  • Оплата в натуральной форме,
  • Доход в натуральной форме,
  • Материальная помощь.

В выше указанных видах расчетов добавлено ограничение на установку признака Рассчитывать результат методом "от обратного". Выводится соответствующее сообщение.

Расчет методом "от обратного" не предусмотрен для начислений, рассчитываемых в соответствии с законодательством РК (для которых невозможна установка произвольной формулы расчета):

  • Оплата отпуска,
  • Компенсация отпуска,
  • Отпуск без оплаты,
  • Неявка по болезни,
  • Неявка по невыясненным причинам,
  • Прогул,
  • Простой по вине работника,
  • Доплата до среднего заработка,
  • Оплата больничного листа за счет работодателя,
  • Отпуск по уходу за ребенком до трех лет,
  • Выходное пособие,
  • Отпуск по беременности и родам,
  • Компенсация за задержку зарплаты,
  • Оплата по договорам подряда,
  • Отгул,
  • Оплата предыдущими документами.
