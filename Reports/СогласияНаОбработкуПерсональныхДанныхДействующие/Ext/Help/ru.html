Согласия на обработку персональных данных действующие

Предназначен для контроля сроков действия согласий на обработку ПДн.

Отображается список действующих согласий с отбором по организации. Возможен отбор по дате действия.

Отражаются Субъект, Дата получения, Срок действия.

При необходимости можно настроить другие условия отбора с помощью кнопки Настройки.
