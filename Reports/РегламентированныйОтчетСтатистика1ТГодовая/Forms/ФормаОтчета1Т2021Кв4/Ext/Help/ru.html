Раздел 2 (Укажите данные о численности работников в среднем за отчетный год и фонде заработной платы)

Раздел заполняется по видам экономической деятельности. Вид экономической деятельности устанавливается в целом для организации на закладке Коды. В случае, если для подразделения вид экономической деятельности отличается, он задается непосредственно в карточке подразделения (поле ОКЭД).

В колонке Списочная численность работников, человек указывается информация о списочной численности работников в среднем за отчетный период. Информация собирается на основании данных регистра сведений Кадровая история сотрудников по виду занятости Основное место работы и регистра сведений Состояния сотрудников.

Списочная численность работников - численность лиц, принятых по трудовому договору, независимо от срока его заключения, кроме лиц, выполняющих работы по совместительству и по договорам гражданско-правового характера.

Для определения списочной численности работников, в среднем за определенный период, ведется ежедневный учет численности работников списочного состава, который уточняется на основании актов работодателя.

Среднесписочная численность (далее С/С) работников определяется путем суммирования среднесписочной численности работников за все месяцы отчетного периода и деления полученной суммы на количество месяцев отчетного периода.

С/С за месяц рассчитывается по формуле:

С/С численность за месяц = ∑С дн / К дн, где ∑С дн – cумма С/С работников за каждый календарный день месяца, К дн – число календарных дней месяца.

С/С за год рассчитывается по формуле:

С/С численность за год = (∑С/С мес.) / 12, где ∑С/С мес – сумма С/С работников за все месяцы отчетного года.

Проверить строку можно с помощью отчета Средняя численность сотрудников организации по графе Списочная численность.

В колонке Фактическая численность работников, человек указывается информация о численности работников, которая определяется согласно "Инструкции по заполнению статистической формы общегосударственного статистического наблюдения".

Фактическая численность работников (принимаемая для исчисления средней заработной платы) – численность работников списочного состава за вычетом отдельных категорий работников.

Исключаются следующие категории работников:

  • обучающиеся в организациях образования и находящиеся в отпусках без сохранения заработной платы, а также работники, поступающие в организации образования и находящиеся в отпуске без сохранения заработной платы для сдачи вступительных экзаменов по соглашению сторон трудового договора, на основании заявления работника;
  • направленные в командировки длительного характера, в том числе в целях выполнения строительных, монтажных и наладочных работ, если за ними не сохраняется заработная плата;
  • находящиеся в отпусках по беременности и родам, в отпусках в связи с усыновлением (удочерением) новорожденного ребенка (детей), в отпусках без сохранения заработной платы по уходу за ребенком до достижения им возраста трех лет;
  • находящиеся в отпусках без сохранения заработной платы по соглашению сторон трудового договора на основании заявления работника

Работники, принятые по совместительству, либо принятые (переведенные) на неполное рабочее время в фактической численности работников учитываются расчетно, и их численность определяется путем деления числа отработанных человеко-часов в отчетном месяце на установленную продолжительность рабочего времени за месяц.

Информация собирается по данным регистра сведений Кадровая история сотрудников по виду занятости Основное место работы и регистру сведений Состояния сотрудников.

Проверить можно с помощью отчета Средняя численность сотрудников организации путем суммирования колонок Фактическая численность, Совместителей по норме времени и По сокращенному графику.

В колонке Фонд заработной платы работников, тысяч тенге указывается информация о фонде заработной платы, начисленной сотрудникам за отчетный период.

В фонде заработной платы учитываются все выплаты, как в денежной, так и в натуральной форме, переведенные в денежную единицу.

Данные о фонде заработной платы собираются и заполняются автоматически только для тех видов расчета, для которых заполнено поле Категория фонда заработной платы для стат формы 1Т годовая на закладке Налоги, взносы, бухучет. Если он не заполнен, а начисление по такому виду расчета выполнено, данные не попадут в фонд начисленной заработной платы.

Проверить колонку можно с помощью отчета Средняя численность сотрудников, добавив колонку Начисления(для 1Т - годовая).

В колонке Среднемесячная номинальная заработная плата одного работника, тенге указывается информация о среднемесячной номинальной заработной плате одного работника, определяемая путем деления суммы начисленного фонда заработной платы на фактическую численность работников и на количество месяцев в отчетном периоде (колонка 5 * 1000 / колонка 3 / 12 или колонка 6 * 1000 / колонка 4 / 12 соответственно).

Раздел 2.1 (Укажите данные о численности работников в среднем за отчетный год и фонде заработной платы административного и производственного персонала)

Информация собирается по данным регистра сведений Статистический учет по признаку Административно-управленческий персонал. Признак устанавливается для должности на закладке Статистический учет. Если включено штатное расписание, берется из справочника Штатное расписание, иначе из справочника Должности.

Строка 1 "Всего по организации (подразделению)" является итоговой расчетной строкой.

Строка 1.1 "административный персонал". Данные заполняются, если установлен признак Административно-управленческий персонал.

Строка 1.2 "производственный персонал". К производственному персоналу относятся остальные работники за минусом административного персонала.

Раздел 3 (Укажите данные о численности работников в среднем за отчетный год и фонде заработной платы по основным группам занятий (согласно Национальному классификатору Республики Казахстан "Классификатор занятий"))

Строка 1 - "Всего по организации (подразделению)" - итоговая по строкам 1.1-1.10

Строка 1.1-1.9 - определяются по основным группам занятий сотрудников. Группа занятий определяется по первому уровню кода занятия. Для заполнения строк в должностях необходимо на закладке Статистический учет заполнить поле. Если включено штатное расписание, код занятий берется из справочника Штатное расписание, иначе из справочника Должности

Строка 1.10 "Работники, не входящие в другие группы". Строка заполняется если не заполнено поле Код занятий в справочнике Должности или в справочнике Штатное расписание в разделе Статистический учет.

Раздел 4 (Укажите данные о численности и фонде заработной платы: лиц, выполняющих работы по договорам гражданско-правового характера; работников, работающих неполное рабочее время и принятых на работу по совместительству, в среднем за отчетный год)

Строка 1 "Численность работников, принятых на работу по совместительству (из других организаций), человек"

В строке выводятся данные о списочной численности работников предприятия, принятых в организацию по совместительству. Информация собирается по данным регистра сведений Кадровая история сотрудников по виду занятости Совместительство. Проверить данные можно с помощью отчета Кадровая история сотрудников.

Строка 2 "Численность лиц, выполняющих работы по договорам гражданско-правового характера, человек"

Данные по ГПХ заполняются, если за отчетный период зарегистрированы начисления с видом дохода Доход от налогового агента.

Строка 3 "Фонд заработной платы, начисленный работникам, принятым по совместительству (из других организаций), тысяч тенге (с десятичным знаком)" - фонд заработной платы, начисленный работникам, принятым по совместительству.

Строка 4 "Фонд заработной платы, начисленный лицам, выполняющих работы по договорам гражданско-правового характера, тысяч тенге (с десятичным знаком)" - фонд заработной платы, начисленный сотрудникам, выполняющим работы по договору ГПХ.

Строка 5 "Численность работающих неполный рабочий день или неполную рабочую неделю"

В строке указываются данные о численности сотрудников, работающих неполный рабочий день или неполную рабочую неделю. Заполняется в случае, если для организации настроена работа по неполной ставке (Настройка - Кадровый учет - Используется работа по неполной ставке). Данные собираются по регистру сведений График работы сотрудников для графиков, у которых установлен признак Неполное рабочее время.

Строка 6 "Численность работников, временно неработающих в связи с простоем производства"

В строке указывается численность работников, временно неработающих в связи с простоем производства. Информация собирается по данным регистров сведений Кадровая история сотрудников, Состояния сотрудников. Если один и тот же работник в течение отчетного периода более одного раза временно не работал в связи с простоем производства, то он показывается один раз за отчетный период. Данную строку можно проверить отчетом Средняя численность сотрудников организации при выводе дополнительной колонки Простои.

Строка 7 "Численность работающих на дистанционной работе, человек"

В строке выводятся данные о фактической численности различных физических лиц, работающих на дистанционной работе.
Исключаются следующие категории работников:

  • обучающиеся в организациях образования и находящиеся в отпусках без сохранения заработной платы, а также работники, поступающие в организации образования и находящиеся в отпуске без сохранения заработной платы для сдачи вступительных экзаменов по соглашению сторон трудового договора, на основании заявления работника;
  • направленные в командировки длительного характера, в том числе в целях выполнения строительных, монтажных и наладочных работ, если за ними не сохраняется заработная плата;
  • находящиеся в отпусках по беременности и родам, в отпусках в связи с усыновлением (удочерением) новорожденного ребенка (детей), в отпусках без сохранения заработной платы по уходу за ребенком до достижения им возраста трех лет;
  • находящиеся в отпусках без сохранения заработной платы по соглашению сторон трудового договора на основании заявления работника.

Проверить строку можно с помощью отчета Средняя численность сотрудников организации по графе Работников дистанционно.

Численность работающих на дистанционной работе за год рассчитывается по формуле:

Фактическая численность работающих на дистанционной работе за год = (∑факт.численность работающих на дистанционной работе мес.) / 12, где ∑факт.численность работающих на дистанционной работе мес. – сумма фактическая численность работников за все месяцы отчетного года.

Раздел 5 (Укажите данные об использовании календарного фонда времени работников)

Строка 1, 2. Число отработанных человеко-дней всеми работниками и Число отработанных человеко-часов всеми работниками, тысяч человеко-часов

В строках выводятся данные об отработанном времени за отчетный период всеми работниками организации. При заполнении данных учитывается фактически отработанное всеми работниками предприятия время. Отработанное время учитывается при условии, что в настройках видов расчета на закладке Учет времени установлен один из признаков: За работу полную смену в пределах нормы, За работу неполную смену в пределах нормы времени, За работу сверх нормы времени.

Строка 3.1 "оплачиваемые ежегодные трудовые отпуска (включая дополнительные трудовые отпуска)"
Строка заполняется данными о количестве дней, в которые работники находились в оплачиваемом ежегодном трудовом отпуске (включая дополнительные трудовые отпуска).

Строка 3.2 "учебные отпуска"- заполняется данными о количестве дней, в которые работники находились в учебных отпусках, полученными по состоянию работников Отпуск учебный оплачиваемый и Отпуск учебный неоплачиваемый.

Строка 3.3 "по болезни" - заполняется данными о количестве дней-неявок по болезни, включающие только рабочие дни в период болезни (без выходных и праздничных нерабочих дней), независимо от того, оплачены эти дни или нет.
Дни неявки определяются по состоянию работников:

  • Болезнь
  • Болезнь без оплаты

Строка 3.4 "в связи с отпусками без сохранения заработной платы" - заполняется данными о количестве дней-неявок в связи с отпусками без сохранения заработной платы по соглашению сторон трудового договора на основании заявления работника.
Дни неявки определяются по состоянию работников:

  • Отпуск неоплачиваемый по законодательству
  • Отпуск неоплачиваемый по разрешению работодателя
  • Дополнительный отпуск (неоплачиваемый)

Строка 3.5 "в связи с простоем производства" - заполняется данными о количестве дней-неявок в связи с простоем производства, включающие человеко-дни простоев работников, которые весь рабочий день (смену) не работали по причинам экономического, технологического, организационного, иного производственного или природного характера и не были временно переведены на другую работу.
Дни неявки определяются по состоянию работников:

  • Простой по вине работника
  • Простой по вине работодателя
  • Простой, не зависящий от работодателя и работника

Строка 3.6 "по другим причинам" - заполняется данными о количестве дней-неявок, разрешенных законодательством, включающие:

  • человеко-дни неявок работников освобожденных временно от работы в связи с выполнением государственных или общественных обязанностей на период одного дня или более;
  • человеко-дни неявок на работу в связи с карантином или уходом за больными, оформленными листами нетрудоспособности выданными в установленном законодательством Республики Казахстан порядке;
  • человеко-дни работников, отвлеченных на устранение последствий стихийных бедствий;
  • человеко-дни неявок на работу работников, находящихся под следствием до решения суда;
  • отпуска в связи с рождением ребенка (детей), усыновлением (удочерением) новорожденного ребенка (детей), которые исчисляются в календарных днях без учета праздничных и выходных дней, приходящихся на дни отпуска и включают отпуска по беременности и родам;
  • отпуска работникам, усыновившим или удочерившим новорожденного ребенка (детей);
  • отпуска без сохранения заработной платы по уходу за ребенком до достижения им возраста трех лет;
  • человеко-дни неявок лиц, занятых на общественных началах (без начисления заработной платы).

Дни неявки определяются по состоянию работников:

  • Командировка
  • Прогул
  • Вынужденный прогул
  • Отпуск по беременности и родам
  • Отпуск по уходу за ребенком
  • Дополнительные выходные дни (оплачиваемые)
  • Дополнительные выходные дни не оплачиваемые
  • Отсутствие по невыясненным причинам
  • Отсутствие с сохранением оплаты

Раздел 6 (Укажите информацию об обучении работников за счет средств работодателя (за отчетный год), человек)

Раздел 7 (Укажите данные о движении рабочей силы, человек)

Раздел заполняется данными о движении рабочей силы, численности работающих неполное время и наличии вакансий.

Строка 1 "Списочная численность работников на начало отчетного периода - всего"

В строке выводятся данные о списочной численности работников предприятия, принятых в организацию на начало отчетного периода. Информация собирается по данным регистра сведений Кадровая история сотрудников.

Проверить данные можно с помощью отчета Кадровая история сотрудников.

Строка 2 "Принято работников за отчетный период"
В строке указывается численность работников, принятых в организацию за отчетный период. Данные собираются по регистру сведений Кадровая история сотрудников по виду события Прием.

Строка 2.1 "специалистов с высшим образованием из числа окончивших высшие учебные заведения в отчетном году"
Заполняется по сотрудникам у которых вид образования Высшее образование — бакалавриат, Высшее образование, Высшее образование - подготовка кадров высшей квалификации и дата окончания в отчетном году.

Строка 2.2 "принято работников на вновь созданные рабочие места" Заполняется по сотрудникам принятых в отчетном году, если ставка по штатному расписанию была создана в отчетном периоде (Дата утверждения штатного расписания в отчетном году)

Строка 2.3 "работников, имеющих инвалидность"
Заполняется по данным регистра сведений Сведения об инвалидности, если сотрудник принят в отчетном периоде

Для заполнения колонок 2,3,4 в карточке сотрудника на закладке Образования, квалификация нужно указать сведения об образовании.

Колонка 2 заполняется по виду образования Среднее профессиональное образование.

Колонка 3 заполняется по видам образования Высшее образование — бакалавриат, Высшее образование, Высшее образование - подготовка кадров высшей квалификации .

Колонка 4 заполняется по видам образования Аспирантура, Ординатура, Адъюнктура, Докторантура, Послевузовское образование, Высшее образование - специалитет, магистратура.

Строка 3 "Выбыло работников за отчетный период — всего"
Является расчетной строкой строк 3.1-3.7

Строки 3.1 - 3.6
В строках выводится численность сотрудников, выбывших за отчетный период в разрезе причин увольнения. Данные собираются по регистру сведений Кадровая история сотрудников по виду события Уволен, а также по указанной в основании увольнения причине увольнения документов Увольнение или Увольнение списком.

Проверить строки 3.1-3.6 можно с помощью отчета Средняя численность сотрудников организации при выводе дополнительных полей:

  • Выбыло за нарушение трудовой дисциплины
  • Выбыло за несоответствие должности
  • Выбыло из-за ликвидации предприятия
  • Выбыло по другим причинам
  • Выбыло по обстоятельствам не зависящим от воли сторон
  • Выбыло по собственному желанию
  • Выбыло по сокращению численности

Строка 3.7
Если основание увольнения не заполнено, данные будут отнесены в строку 3.7.

Строка 4 "Списочная численность работников на конец отчетного периода"
В строке указывается списочная численность работников на конец отчетного периода.
Данная строка рассчитывается по следующей формуле:
Строка 4 = Строка 1 + Строка 2 - Строка 3.

Раздел 8 (Укажите данные о составе списочной численности работников на конец отчетного года, человек)

Заполняется по списочной численности работников на конец отчетного года.

Строка 1 "Всего по организации (подразделению)"- итоговая по строкам 1.1 - 1.4.

Строки 1.1-1.4 — Заполняются из регистра сведений Кадровая история сотрудников по датам рождения сотрудников.

Строка 2 "работающие пенсионеры"- определяется по данным регистра сведений Сведения о пенсионном обеспечении физических лиц.

Строка 3 "работники, имеющие инвалидность"- определяется по данным регистра сведений Сведения об инвалидности физических лиц.

Строка 3.1 "из них женщины, имеющие инвалидность"- определяется по данным регистра сведений Сведения об инвалидности физических лиц.

Строка 4 "численность работающих на "зеленых рабочих местах" - заполняется списочной численностью работающих на "зеленых рабочих местах". К "зеленым рабочим местам" относятся рабочие места, способствующие сохранению и восстановлению окружающей среды (ОКЭД – 36, 37, 38, 39, 74.90.1).
