Код БИН - определяется по данным из карточки организации (юр. лица) из закладки Главное поля ИИН/БИН, по которой заполняется отчет.

Раздел 1 (Данные по юридическому лицу)

Фактическое место расположения юридического лица – определяется из карточки организации (юр. лица), закладки Адреса и телефоны, по которой заполняется отчет.
Код КАТО - определяется по данным регистра сведений Адресные объекты, страны, в которой зарегистрирована организация (юр. лицо).
Наименование экономической деятельности – определяется по данным карточки организации (юр. лица), закладки Коды – поле ОКЭД.

Раздел 2 (Фактическая численность работников в среднем за отчетный период, человек)

Раздел заполняется по фактической численности работников, которая определяется согласно "Инструкции по заполнению статистической формы общегосударственного статистического наблюдения".
В отчете заполняются только данные за апрель текущего года.

В строке Фактическая численность работников в среднем за отчетный период, человек учитывается численность работников списочного состава за вычетом отдельных категорий работников, имеющих формальное прикрепление к работе (например: за исключением сотрудников, находящиеся в декретном отпуске). Сотрудники, находящиеся в отпуске, из отчета не исключаются.
Также в отчет не попадают физические лица, работающие по договору гражданско-правового характера.

Проверить заполнение строки можно с помощью отчета Средняя численность сотрудников организации.

Раздел 3 (Данные по работникам юридического лица)

Для составления раздела, формируется список сотрудников для выборочного обследования по установленным условиям.
Условия определяются исходя из фактической численности (фактическая численность определяется в разделе 2).
В список для выборочного обследования входят только те сотрудники, отработавшие полный и не полный месяц, за исключением работников находившихся в отпуске с 1 по 30 апреля.

Если сотрудник весь месяц находился в отпуске, он не включается в список для выборочного обследования (данные по нему не отражаются в разделе 3).

Раздел заполняется "выборочно" по установленному алгоритму:
1. Фактическая численность работников в среднем за отчетный период от 1 до 100 человек – отбору подлежат все работники (применяется сплошной учет);
2. Фактическая численность работников в среднем за отчетный период от 101 до 250 человек – отбору подлежит каждый второй работник;
3. Фактическая численность работников в среднем за отчетный период от 251 до 1000 человек – отбору подлежит каждый десятый работник;
4. Более 5000 человек – отбору подлежит каждый пятидесятый работник.

Для отбора работников сортировка производится согласно табельному номеру.

В колонке Табельный номер заполняется информация по табельным номерам сотрудников.
Просмотреть табельный номер можно в справочнике Сотрудники.

В колонке Пол заполняется информация из справочника Физические лица.

В колонке Год рождения заполняется информация из справочника Физические лица.

В колонку Дата приема на работу в юридическое лицо заполняется информация из регистра сведений Кадровая история сотрудников.

В колонке по Классификатору занятий заполняется информация из регистра сведений Статистический учет.
Данные в регистр сведений заполняются из справочника Должности (если отключено ведение штатного расписания), либо если включено ведение штатного расписания, заполняется информация из справочника Штатное расписание.

В колонке Уровень образования (среднее образование из нее общее среднее, техническое и профессиональное, высшее, послевузовское) указывается информация из справочника Физические лица гиперссылка Образование, квалификация.
В отчет заполняются данные по последнему месту обучения.

В колонке Вид оплаты труда(1 - тарифная, 2 - бестарифная) автоматически заполняется значение 1.

В колонке Фонд заработной платы одного работника,тысяч тенге (с десятичным знаком) указывается информация о фонде заработной платы, начисленной сотрудникам за анализируемый период.

Данные о фонде заработной платы собираются и заполняются автоматически для тех видов расчета, у которых на закладке Основное установлен признак Включать в ФОТ, а также для тех видов расчета, у которых заполнено поле Категория фонда заработной платы для форм статистики на закладке Налоги, взносы, бухучет.

При этом, если указана категория фонда заработной платы Оплата за неотработанное время, то сумма в отчет не заполнится.
Также в отчет не заполняются суммы по: разовым единовременным выплатам и
разовым премиям (в данном случае, у вида начисления не должен быть установлен признак Включать в ФОТ и заполнена категория фонда заработной платы).

В колонке Режим работы(1 - полный, 2 - неполный) заполняется информация из графика работы сотрудника.
Неполное рабочее время заполняется в случае, если для организации настроена работа по неполной ставке (Настройка - Кадровый учет - Используется работа по неполной ставке). И в графике работы сотрудников установлен признак Неполное рабочее время.

В колонке Число отработанных часов на одного человека заполняются данные об отработанном времени за анализируемый период.
