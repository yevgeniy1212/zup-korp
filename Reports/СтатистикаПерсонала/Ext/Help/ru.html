Списочная численность

В колонке выводится информация о списочной численности работников. Информация собирается на основании данных регистра сведений Кадровая история сотрудников по виду занятость Основное место работы и регистра сведений Состояния сотрудников

Списочная численность работников - численность лиц, принятых по трудовому договору, независимо от срока его заключения, кроме лиц, выполняющих работы по совместительству и по договорам гражданско-правового характера.

Фактическая численность

Фактическая численность. В колонке выводятся сведения о фактической численности работников организации.

Информация собирается по данным регистра сведений Кадровая история сотрудников по виду занятости Основное место работы и регистру сведений Состояния сотрудников.

Фактическая численность работников – численность работников списочного состава за вычетом отдельных категорий работников.

Исключаются следующие категории работников:

  • находящиеся в отпусках по беременности и родам;
  • находящиеся в отпусках по уходу за ребенком;
  • находящиеся в неоплачиваемых отпусках по законодательству;
  • находящиеся в неоплачиваемых отпусках по разрешению работодателя;
  • находящиеся в дополнительных неоплачиваемых отпусках;
  • находящиеся в неоплачиваемых учебных отпусках.

Совместителей по норме графика. В колонке выводятся сведения о численности работников, работающих в организации по совместительству (как внутреннему, так и внешнему).

Информация собирается по данным регистра сведений Кадровая история сотрудников по видам занятости Совместительство, Внутреннее совместительство и регистру сведений Состояния сотрудников.

Работники, принятые по совместительству, в фактической численности работников учитываются расчетно, и их численность определяется путем деления числа отработанных человеко-часов в отчетном месяце на установленную продолжительность рабочего времени за месяц.

По сокращенному графику. В колонке указываются данные о численности сотрудников, работающих неполный рабочий день или неполную рабочую неделю. Заполняется в случае, если для организации настроена работа по неполной ставке (Настройка - Кадровый учет - Используется работа по неполной ставке). Данные собираются по регистру сведений График работы сотрудников из справочника Графики работы сотрудников для графиков, у которых установлен признак Неполное рабочее время.

Работники, принятые (переведенные) на неполное рабочее время в фактической численности работников учитываются расчетно, и их численность определяется путем деления числа отработанных человеко-часов в отчетном месяце на установленную продолжительность рабочего времени за месяц.

Работники, для которых зарегистрированы отклонения (отсутствия) в фактической численности работников учитываются расчетно, и их численность определяется путем деления числа отработанных человеко-часов в отчетном месяце на установленную продолжительность рабочего времени за месяц.

Численность работников по договорам ГПХ

Указываются сведения о численности работников, работающих в организации по договорам гражданско-правового характера.

Количество дней простоя

По вине работника. Указывается информация о количестве дней простоя. Данные собираются по регистру сведений Состояния сотрудников организации с состоянием Простой, по вине работника.

По вине работодателя. Указывается информация о количестве дней простоя. Данные собираются по регистру сведений Состояния сотрудников организации с состоянием Простой, по вине работодателя.

Прочие причины. Указывается информация о количестве дней простоя. Данные собираются по регистру сведений Состояния сотрудников организации с состоянием Простой, не зависящий от работодателя и работника.

ФОТ (Фонд оплаты труда)

К фонду заработной платы (оплаты труда) относятся начисленные суммарные денежные средства организации для оплаты труда работников (должностные оклады (тарифные ставки), доплаты, надбавки, премии и иные выплаты стимулирующего и компенсирующего характера), с учетом налогов и других удержаний (подоходный налог, обязательные пенсионные взносы) в соответствии с нормативными правовыми актами Республики Казахстан и независимо от источника их финансирования и срока их фактических выплат. В фонде заработной платы учитываются все выплаты как в денежной, так и в натуральной форме, переведенные в денежную единицу.

Данные о фонде заработной платы собираются и заполняются автоматически только для тех видов расчета, для которых установлен признак Включать в ФОТ на закладке Основное. Если он не установлен, а начисление по такому виду расчета выполнено, данные не попадут в фонд начисленной заработной платы.

Списочный состав. Указывается информация по фонду заработной платы списочного состава сотрудников организации.

Внешние совместители. Указывается информация о фонде заработной платы сотрудников, работающих в организации по совместительству.

Внутренние совместители. Указывается информация о фонде заработной платы сотрудников, работающих в организации по внутреннему совместительству.

Договора ГПХ. Указывается информация о фонде заработной платы сотрудников, работающих в организации по договорам гражданско-правового характера.

Отработанное время

Дней. Часов. Указываются данные об отработанном времени за отчетный период всеми работниками организации. При заполнении данных учитывается фактически отработанное всеми работниками предприятия время. Отработанное время учитывается при условии, что в настройках видов расчета на закладке Учет времени установлен один из признаков: За работу полную смену в пределах нормы, За работу неполную смену в пределах нормы времени, За работу сверх нормы времени.
