При открытии отчета необходимо указать налогоплательщика и налоговый комитет, в который будет предоставлен отчет. Если код налогового органа структурного подразделения совпадает с кодом налогового органа налогоплательщика, то данные по структурному подразделению тоже будут отражены в форме отчета.

Автоматическое заполнение графы в конфигурации не предусмотрено.

Заполнение графы X (Доходы, облагаемые социальным налогом)

Графа заполняется суммами из колонки Объект исчисления регистра накопления Сведения об исчислении налогов, взносов, отчислений.

Заполнение графы Y (Социальный налог, исчисленный с начисленных доходов)

Графа заполняется суммами социального налога, исчисленного по физическому лицу (до исключения СО), за месяцы налогового периода, входящие в отчетный период, согласно данным регистра накопления Сведения об исчислении налогов взносов отчислений по виду налога Социальный налог.

Заполнение графы Z (Сумма отчислений на ГФСС)

Графа заполняется суммами социальных отчислений, исчисленного по физическому лицу, за месяцы налогового периода, входящие в отчетный период, согласно данным регистра накопления Сведения об исчислении налогов взносов отчислений по виду налога Социальные отчисления.

Заполнение графы AA (Социальный налог, подлежащий уплате в бюджет)

Графа заполняется суммами социального налога, исчисленного по физическому лицу (с учетом исключения СО), за месяцы налогового периода, входящие в отчетный период, согласно данным регистра накопления Сведения об исчислении налогов взносов отчислений по виду налога Социальный налог.

Заполнение графы AB (Сумма обязательных профессиональные пенсионных взносов, подлежащих уплате)

Графа заполняется суммой обязательных профессиональных пенсионных взносов, исчисленных с доходов физлица в соответствии с пенсионным законодательством Республики Казахстан, рассчитанные документом Начисление зарплаты (на основании данных регистра Сведения об исчислении налогов взносов отчислений по виду налога Обязательные профессиональные пенсионные взносы).

Заполнение графы AC (Сумма отчислений на ОСМС, подлежащих уплате)

В строке выводятся данные о суммах отчислений на обязательное социальное медицинское страхование, исчисленных за соответствующий месяц налогового периода, определяемые по регистру накопления Сведения об исчислении налогов взносов отчислений по виду налога Отчисления на ОСМС.
