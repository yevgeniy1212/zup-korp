Структура задолженности взносов и отчислений

Отчет предназначен для получения сведений о задолженности организации по взносам и отчислениям в разрезе вида взноса/отчисления.
Для формирования отчета обязательно указывается вид взноса/отчисления, также предусмотрен отбор по организации и физическому лицу.
При включенной настройке По месяцу налогового периода отчет собирается по месяцу налогового периода взноса/отчисления.
Колонка Сальдо на начало, при включенной настройке, всегда заполняется нулевым значением. Такое поведение обусловлено отсутствием задолженности по месяцу налогового периода на его начало. Колонка Сальдо конечное заполняется только в тех месяцах, где на конец месяца есть задолженность.
При отключенной настройке отчет собирается по фактическому периоду возникновения задолженности и перечисления.

Поставляются следующие варианты отчеты:

  • Структура задолженности организации (по месяцам);
  • Структура задолженности организации (по физическим лицам).

Если в настройках учетной политики установлен признак Отражать в НУ удержание налогов и взносов пропорционально выплаченному доходу колонка К перечислению по ОПВ и ВОСМС заполняется только при наличии проведенного документа Удержание налогов и взносов по налоговому учету в период формирования отчета.
