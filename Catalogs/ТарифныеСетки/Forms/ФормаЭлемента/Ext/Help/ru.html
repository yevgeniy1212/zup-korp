Тарифная группа

В рамках тарифной группы можно установить для разрядов (категорий) размеры тарифов, которые соответствуют этим разрядам.

Утвержденную тарифную группу впоследствии можно использовать при описании должностей или позиций штатного расписания.

Тогда при приеме сотрудника на эту должность (позицию штатного расписания) будет предложено указать его разряд (категорию) из состава разрядов выбранной тарифной группы.

При заведении новой тарифной группы вводится краткое и полное наименование. Полное наименование используется для вывода в печатные формы, просто наименование – для представления внутри программы. Добавляются разряды (категории), входящие в данную группу и устанавливаются тарифы. Категории выбираются из справочника Квалификационные разряды(категории).

Также можно указать, что размеры тарифов устанавливаются в привязке к базовому тарифу. В этом случае при установке размеров тарифов (утверждении группы) достаточно будет указать только базовый тариф, а тарифы для всех разрядов (категорий) будут определены умножением указанных для них коэффициентов на базовый тариф.

При записи группы создается документ Утверждение тарифной группы, доступный с рабочего стола меню Зарплата.

Если зайти в тарифную группу, которая уже был создана, можно установить новые тарифы разрядов(категорий) по одноименной ссылке. Будет создан новый документ Утверждение тарифной группы, где можно указать новые тарифы и проконтролировать изменения начислений штатного расписания и ФОТ.

Примечание. Создавать и описывать новые группы, а также вводить их изменения можно и непосредственно документами Утверждение тарифной группы, минуя справочник Тарифные группы.
