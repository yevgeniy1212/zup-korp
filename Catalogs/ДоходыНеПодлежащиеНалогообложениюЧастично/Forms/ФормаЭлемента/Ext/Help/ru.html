Доход не подлежащий налогобложению частично

В программе предусмотрена возможность регистрации доходов, которые не подлежат налогообложению частично.

Список доходов

В поставке программы предусмотрен набор популярных доходов. Минимальный набор доходов, соответствующий базовым возможностям программы.

Имеется возможность добавить доходы непосредственно в список, указать в карточке соответствующие данные, необходимые для расчета.

Карточка дохода, подлежащего налогообложению частично

Предназначена для указания основных параметров дохода, по которым будет выполняться рассчет.

В поле Наименование указывается наименование дохода, по которому его можно определить в списке.

В поле Размер облагаемого дохода (количество МЗП) указывается количество минимальных заработных плат, которые не будут облагаться налогами и взносами по зарегистрированному доходу сотруднику организации. Суммы, превышающие указанный предел, подлежат налогообложению.

В поле Период предоставления выбирается период, в течении которого указанный предел не будет подлежать налогообложению частично:

  • В пределах месяца;
  • В пределах календарного года.

Крыж Не используется устанавливается для тех доходов, которые больше не используются в программе. После его установки в списке доходов он будет скрыт.
