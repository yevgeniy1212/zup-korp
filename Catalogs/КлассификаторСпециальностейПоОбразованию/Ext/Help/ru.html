Специальности (ОКСО)

Справочник может быть заполнен по данным классификатора специальностей по образованию. Для этого предназначена кнопка Подбор из ОКСО над списком. При ее нажатии открывается классификатор, в котором можно найти и подобрать в справочник требуемые специальности. Данные справочника используются при вводе сведений об образовании сотрудников.
