Национальности народов мира

Справочник содержит перечень национальностей. Используется для заполнения значения реквизита Национальность справочника Физические лица.

Справочник может быть заполнен по данным из классификатора национальностей народов мира. Для этого в форме списка имеется кнопка Подбор из классификатора. При этом открывается список классификатора, в котором можно найти, выбрать и занести в справочник нужную национальность.
