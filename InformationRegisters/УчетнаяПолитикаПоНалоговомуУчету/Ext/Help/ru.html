Система налогообложения (действует с 2021 г.)

Организация является агентом по уплате ОППВ

Данный признак необходимо установить, если организация является вкладчиком обязательных профессиональных пенсионных взносов в пользу работников, занятых на работах с вредными (особо вредными) условиями труда.

Применять ограничение на минимальный объект отчислений ООСМС

Если установлен флажок, то при расчете отчислений ОСМС, если объект исчисления меньше МЗП, в качестве объекта будет учитываться МЗП. Действует с 01.01.2018 г.

Предоставлять разрешенный вычет в месяце отсутствия дохода

Если установлен флажок, разрешенный вычет будет предоставляться в месяце отсутствия дохода. Действует с 01.01.2018 г.

Отражать в НУ удержание налогов и взносов пропорционально выплаченному доходу

Суммы налогов и взносов, подлежащие перечислению, будут рассчитываться от выплаченных доходов работника.
При установленном флажке суммы налогов и взносов, подлежащие перечислению, будут рассчитываться с помощью документа Удержание налогов и взносов по налоговому учету.

Применять вычет ОПВ для ГПХ (действует до 2020 г.)

Действует с 01.01.2019 по 31.12.2019. Законом Республики Казахстан № 382-VI "О внесении изменений и дополнений в Кодекс Республики Казахстан "О налогах и других обязательных платежах в бюджет" (Налоговый кодекс) предусмотрено предоставление физическим лицам по договорам ГПХ вычета по ОПВ при расчете ИПН.
Если признак установлен, при расчете ИПН предоставляется вычет в размере ОПВ.
Если признак не установлен, вычет по ОПВ не предоставляется.

Расчет СН, ООСМС, ВОСМС без применения 90 % корректировки по ИПН (действует в 2020 г.)

Действует с 01.01.2020 по 31.12.2020. Законом Республики Казахстан № 382-VI "О внесении изменений и дополнений в Кодекс Республики Казахстан "О налогах и других обязательных платежах в бюджет" (Налоговый кодекс) предусмотрен расчет СН, ООСМС, ВОСМС без применения 90% корректировки по ИПН.
Если признак установлен, то при расчете социального налога, отчислений и взносов на обязательное социальное медицинское страхование не применяется 90% корректировка облагаемого дохода работников в соответствии с пп. 49) пункта 1 статьи 341 НК РК.
Если признак снят, то корректировка участвует в расчете социального налога и взносов/отчислений по ОСМС.

Применять вычет ВОСМС для физических лиц (сотрудников) (действует в 2020 г.)

Действует с 01.01.2020 по 31.12.2020. Законом Республики Казахстан № 382-VI "О внесении изменений и дополнений в Кодекс Республики Казахстан "О налогах и других обязательных платежах в бюджет" (Налоговый кодекс) предусмотрено предоставление физическим лицам вычета по ИПН:
- по доходам работников вычета на сумму ВОСМС.
Если признак установлен, то при расчете ИПН физическим лицам предоставляются вычеты.
Если признак снят, то вычеты не предоставляются.

Применять вычеты ОПВ и ВОСМС для ГПХ (действует в 2020 г.)

Действует с 01.01.2020 по 31.12.2020. Законом Республики Казахстан № 382-VI "О внесении изменений и дополнений в Кодекс Республики Казахстан "О налогах и других обязательных платежах в бюджет" (Налоговый кодекс) предусмотрено предоставление физическим лицам вычетов по ИПН:
- по доходам гражданско-правового характера вычетов на сумму ОПВ;
- по доходам гражданско-правового характера вычетов на сумму ВОСМС.
Если признак установлен, то при расчете ИПН физическим лицам, работающим по договору ГПХ, предоставляются вычеты.
Если признак снят, то вычеты не предоставляются.

Система налогообложения (действует с 2021 г.)

Расчет налогов, взносов, отчислений будет производится в соответствии с установленной системой налогообложения.

Для индивидуальных предпринимателей, использующих общеустановленный режим налогообложения (ОУР) доступны для заполнения реквизиты:

  • Сумма заявленного дохода ИП - доход для исчисления ОПВ и СО;
  • Коэффициент МРП для расчета СН за ИП - используется для расчета СН по доходу индивидуального предпринимателя;
  • Коэффициент МРП для расчета СН - используется для расчета СН по доходу от работодателя;
  • Способ отражения по доходу субъекта предпринимательства - указывается отражение в регламентированном учете.
