Упорядочивание списка сотрудников в документах

По умолчанию в документах, которые заполняются списком сотрудников автоматически, сотрудники упорядочиваются по "приоритету" их позиции, т.е. по порядку подразделения и должности, а уже потом по имени.

Настройка упорядочивания в этой форме позволяет установить единый порядок сортировки по умолчанию сразу для всех документов и пользователей, если привычней и удобней просматривать списки в каком-то другом порядке, например, по имени. Это позволяет не прибегать к настройке сортировки в каждом экземпляре документа.

Установленный порядок будет применяться практически ко всем документам по умолчанию, кроме некоторых документов, которые заполняются списком физических лиц, а не сотрудников.

Настройки печатных форм

Выводить полные ФИО в списочных печатных формах

Если настройка не установлена, то в списочных печатных формах имя сотрудников будет выводится в виде "Иванов И.И.".
Если требуется выводить ФИО полностью, то можно установить эту настройку, и тогда имя в таких формах будет выводиться в виде "Иванов Иван Иванович".

Удалять префиксы организации и ИБ из номера кадровых приказов

По умолчанию при формировании печатной формы приказов, например, о приеме на работу из номера документа автоматически получается только значимая часть. Например, если документ программы имеет номер ПОПБ-000256, то в печатной форме в качестве номера приказа будет выведен номер 256. Однако если выключена эта настройка, то номер будет выведен в приказ полностью. Эта настройка может быть использована и в ситуации, когда ведется собственная нестандартная нумерация приказов, и в результате удаления префиксов номер в приказ не попадает совсем.

Выводить полную иерархию подразделений

Если сотрудник оформляется в подразделение, у которого есть вышестоящие подразделения, то по умолчанию в приказ, например, о приеме на работу, будет выведено название только самого подразделения (конечного). Однако если включена эта настройка, то в приказ будет выведена вся цепочка подразделений, например "Управление установки и эксплуатации оборудования/Отдел эксплуатации ЛВС и средств связи" вместо "Отдел эксплуатации ЛВС и средств связи".
