Начисление зарплаты и взносов

Документ выполняет окончательный расчет зарплаты по итогам каждого месяца. Его следует вводить после ввода всех первичных документов.
В поле Месяц указывается месяц, в котором будут производиться взаиморасчеты с сотрудниками, расчет налогов, отражение в учете (по умолчанию – текущий месяц).
Для автоматического заполнения и расчета документа по всем сотрудникам предназначена кнопка Заполнить - Заполнить.
Если необходимо начислить зарплату конкретным сотрудникам используется кнопка Подбор.
Документ можно перезаполнить, сохранив вручную исправления по кнопке Заполнить - Заполнить, сохранив исправления.
Например, если прием на работу введен уже после заполнения документа. При этом по уже заполненным сотрудникам перезаполнение не производится можно дозаполнить документ по кнопке Заполнить - Дозаполнить новыми сотрудниками.
