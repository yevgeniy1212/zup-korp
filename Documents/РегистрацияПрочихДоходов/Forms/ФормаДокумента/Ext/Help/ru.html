Начисление прочих доходов

Документ предназначен для регистрации выплат различных доходов вне рамок трудовых отношений. Он доступен, если в настройках расчета зарплаты выбрана соответствующая возможность.

По зарегистрированным суммам может образовываться задолженность организации перед получателями таких доходов. Предполагается, что все денежные расчеты вне рамок трудовых отношений производятся в бухгалтерской программе (подсистеме).

Если требуется выполнять расчеты с сотрудниками по прочим доходам так же, как расчеты по оплате труда, то их можно регистрировать, например, как произвольное разовое начисление.

См. также

- Для регистрации договоров на выполнение работ или оказание услуг, а также договоров авторского заказа могут также использоваться отдельные документы.

Заполнение документа

Указанным в документе месяцем суммы будут отражены в бух. учете.

Вид дохода выбирается из списка, который можно пополнять самостоятельно.
Способ обложения заполняется в соответствии с выбранным видом дохода и отражается в документе только для просмотра. При необходимости их можно изменить в карточке вида дохода.
Для регистрации выплат с целью исчисления налогов и взносов за физических лиц, получающих доходы по договорам гражданско-правового характера необходимо указывать вид дохода - Доходы налогового агента.
Для регистрации доходов ИП с целью исчисления налогов,взносов, отчислений необходимо указывать вид дохода - Доходы субъекта предпринимательства.

В документе можно уточнить способ отражения регистрируемых им сумм в учете.

Также можно указать подразделение, которое будет использоваться при отражении сумм в бух. учете.

При подборе (выборе) физ. лиц в таблицу документа по умолчанию предлагаются только те, которым уже ранее регистрировались прочие доходы. Если требуемое лицо отсутствует в предложенном списке, то можно его либо добавить, либо найти в полном списке физ. лиц.

Сумма дохода указывается в таблице, при этом автоматически рассчитываются налоги и взносы.
