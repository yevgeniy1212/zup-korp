Утверждение тарифной группы

Документ предназначен для описания тарифной группы, т.е. перечисления составляющих ее разрядов (категорий) и установки размера тарифа для каждого из них.

Он служит как для первоначально утверждения, так и для последующего изменения тарифов. При этом если числятся работающие сотрудники с разрядами из изменяемой тарифной группы, то в этом же документе производится изменение размера их плановых начислений и ФОТ. Корректируется также и ФОТ позиций штатного расписания, в которых выбрана эта тарифная группа.

Заполнение документа

В документе указывается тарифная группа, разряды которой требуется утвердить или изменить. Документ удобно вводить из карточки тарифной группы. В этом случае она заполняется автоматически.

Дата и номер документа используются в печатной форме приказа об изменении тарифной группы.

Размеры тарифов, указанные в документе, вступают в силу с указанной в одноименной поле даты.

Если для тарифной группы указано, что размеры тарифов устанавливаются в привязке к базовому тарифу, то этот тариф обязательно следует указать в поле базовый тариф. Исходя из него будут определяться размеры тарифов для составляющих группу разрядов.
В противном случае базовый тариф указывать можно, но не обязательно. Согласно введенному базовому тарифу определяется значение показателя расчета зарплаты Базовый тариф группы, который можно использовать при описании начислений.

На закладке Тарифы перечисляются разряды (категории), составляющие эту тарифную группу. Разряды выбираются из соответствующего справочника. Если для описания группы имеющихся в поставке программы разрядов недостаточно, то их следует добавить в этот справочник.

Для каждого разряда указывается либо коэффициент, который будет применяться к базовому тарифу, либо непосредственно размер тарифа, в зависимости от вида тарифной группы (установлено ли для нее применение тарифных коэффициентов или нет).

После проведения документа из него можно сформировать печатную форму приказа об изменении тарифной группы.
