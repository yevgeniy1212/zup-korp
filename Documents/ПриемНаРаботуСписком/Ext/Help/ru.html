Прием на работу списком

Документ предназначен для регистрации приема на работу нескольких сотрудников и для оформления печатной формы приказа о приеме.
Работа с каждой строкой документа аналогична работе с документом Прием на работу, который предназначен для оформления только одного сотрудника.

Для оформления в программе приема на работу всех сотрудников, которые уже работают на предприятии на момент начала использования программы, предназначен специальный документ Данные на начало эксплуатации.

При расчете фонда оплаты труда (далее ФОТ) для начислений методом "от обратного" не учитываются статусы и вычеты, назначенные физическому лицу, поскольку они могут изменяться в течение периода работы. Если требуется изменить ФОТ вручную, необходимо в настройках начисления методом "от обратного" снять признак ФОТ рассчитывается автоматически.

Важно! При снятом признаке ФОТ рассчитывается автоматически, суммы ФОТ рассчитываются и вводятся вручную (в том числе при изменении показателей начислений).
Если требуется рассчитать ФОТ автоматически, необходимо установить 0 в колонке Вклад в ФОТ.
