Увольнение списком

Документ предназначен для регистрации увольнения нескольких сотрудников и для оформления соответствующей печатной формы приказа об увольнении.
Работа с каждой строкой документа аналогична работе с документом Увольнение, который предназначен для оформления только одного сотрудника.
