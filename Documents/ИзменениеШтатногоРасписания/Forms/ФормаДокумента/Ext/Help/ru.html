Изменение штатного расписания

Документ предназначен для внесения "локальных" изменений в штатное расписание, не связанных с существенным изменением штатной структуры организации. Его удобно использовать для организаций со сложной структурой (большим количеством подразделений и позиций) при необходимости внести небольшие изменения, например, в состав позиций одного подразделения или условия существующих позиций.
В отличие от документа Утверждение штатного расписания этим документом можно регистрировать изменения с любого числа месяца, а не только с его начала.

Заполнение документа

Вносимые документом в штатное расписание изменения будут действовать с даты, указанной в поле Дата изменений.

В качестве даты документа может быть указана любая дата – дата реального решения об изменении штатного расписания, дата ввода документа в программу и т.п. Она не влияет на учет штатного расписания в программе.

Пока в документе не указано подразделение, его таблицу можно заполнять произвольными позициями. Для изменения существующих позиций их можно подобрать из штатного расписания по кнопке Изменить позицию, для создания новых – Добавить позицию, для закрытия существующих - Закрыть позицию.

Выбрать подразделение можно в том случае, если при настройке программы не была выбрана возможность расчета и выплаты зарплаты по организации в целом.

При указании подразделения документ заполняется его позициями, действующими на дату изменения. Можно изменить условия таких позиций, закрыть их, либо добавить новые позиции в это подразделение.

Примечание. Если требуется закрыть уже подобранную в документ позицию, то следует воспользоваться командой Закрыть позицию контекстного меню, выделив в таблице эту позицию. Также можно выделить и закрыть сразу несколько позиций.

Редактирование сведений о позиции выполняется в ее карточке.

Дополнительные возможности

Сотрудники в программе могут быть оформлены до начала ведения штатного расписания в ней, либо в разрез с утвержденным в программе штатным расписанием.
В таких случаях позиции в документе можно заполнить автоматически по уже введенным в программу кадровым данным с помощью команды Дополнить по текущей кадровой расстановке в меню Еще (или Все действия) таблицы позиций.

Если позиции введены в документ в произвольном порядке, то для удобства визуального восприятия документа таблицу позиций можно упорядочить в соответствии с порядком, настроенным в списках подразделений и должностей, с помощью одноименной кнопки.

При изменении оплаты труда по позиции может потребоваться отразить это изменение на действующих плановых начислениях сотрудников этой позиции. Для этого можно создать документ Изменение плановых начислений с помощью кнопки Изменить плановые начисления.

Проведение документа

При проведении заполненного документа в штатном расписании создаются утвержденные позиции в полном соответствии с их описанием в документе. Дата их утверждения соответствует месяцу, указанному в документе. Для закрытых позиций указывается дата их закрытия. Для подразделений и должностей устанавливаются признаки их введения в штатное расписание или выведения из него и соответствующие даты.
Если отменить проведение документа, то созданные им позиции не удаляются из штатного расписания – они отмечаются как неутвержденные.
