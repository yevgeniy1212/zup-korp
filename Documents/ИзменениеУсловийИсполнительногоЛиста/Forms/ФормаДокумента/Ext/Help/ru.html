Изменение исполнительного листа

Документ предназначен для изменения условий удержания с определенной даты, назначенного документом Исполнительный лист, а также для его прекращения.

Документ удобно вводить на основании исполнительного листа, условия которого требуется изменить.

Заполнение документа

Для выбранного исполнительного листа можно изменить любые условия, кроме получателя.
Порядок заполнения новых условий полностью аналогичен порядку заполнения самого исполнительного листа.
В случае прекращения удержания указывается только дата прекращения. Начиная с этой даты удержания, назначенные выбранным исполнительным листом, при расчете выполняться не будут.

Важно! Если в исполнительном листе, на основании которого вводятся изменения удержаний по ИЛ, установлена предельная сумма удержаний, то при изменении условий исполнительного листа необходимо указывать размер предельной суммы удержаний.
