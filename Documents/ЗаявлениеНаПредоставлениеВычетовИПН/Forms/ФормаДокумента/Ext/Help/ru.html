Заявление на предоставление вычетов ИПН

Документ предназначен для первоначального ввода и последующего изменения сведений о налоговых вычетах, которые будут учитываться при расчете ИПН. Для зарегистрированных сумм прочих вычетов до 2018 года используется форма документа (до 2018 года). Для регистрации сведений о предоставляемых вычетах с 2018 года используется новая форма документа. В новой форме документа не указываются сведения о льготах, о стандартных и прочих вычетах, а заполняются сведения по видам вычетов, предусмотренных законодательством РК.
Прекращение действия отдельных вычетов также регистрируется этим документом.

Сведения, внесенные в документ, будут применяться начиная с указанного в документе месяца налогового периода, указанного в поле Месяц.
Другими словами, в документ следует вносить исчерпывающую информацию о вычетах, которые следует применять с определенного месяца, все предыдущие сведения (введенные другими документам с меньшим месяцем налогового периода) перестают с этого месяца действовать. В связи с этим недопустимо вводить два заявления с одинаковым месяцем налогового периода.
Прекращение применения вычетов устанавливается посредством заполнения поля Действует по. Если значение данного поля будет не заполнено, то вычеты будут применяться без ограничения срока.

Действие документа и его проведение возможно только при установленном признаке Наличие заявления. Данный признак предназначен для подтверждения получения подписанного сотрудником заявления.

Вычеты, для которых предусмотрен график платежей, применяются согласно данных графика и периода его действия. В графике необходимо указать размер и период действия вычета.

Прекращение применения всех действующих вычетов сотрудника (например, если сотрудник собирается получать вычеты в другой организации) регистрируется документом Прекращение предоставления вычетов ИПН.

Просмотр и изменение всех сведений о вычетах сотрудника (в том числе ввод этих документов) можно выполнять из раздела Налоги карточки сотрудника по гиперссылке Все заявления на вычеты.

Выбор организации и сотрудника

Если документ введен из карточки сотрудника, то поля Организация и Сотрудник автоматически заполняются соответствующими сотруднику данными.

Если документ введен из журнала документов, то следует указать организацию, в которой будут предоставляться вычеты, и сотрудника этой организации. При этом если в программе ведется учет только по одной независимой организации, то эта организация заполняется автоматически.

Если же в программе ведется учет по нескольким организациям, перед выбором сотрудника обязательно следует указать организацию, в которой требуется предоставлять вычеты. После выбора организации в списке сотрудников будут доступны для выбора только сотрудники этой организации.

Вычеты регистрируются применительно к физическому лицу в целом и действуют в рамках конкретной организации. Т.е. если лицо занято в одной организации на нескольких местах работы, даже в подразделениях, имеющих различную регистрацию в налоговом органе (ведено несколько элементов в справочник "Сотрудники"), то при выборе сотрудника в заявление в предлагаемом программой списке такому сотруднику будет соответствовать один элемент (физическое лицо).
Таким образом, в программе нельзя:
• на один и тот же период (месяц налогового периода) зарегистрировать сотруднику применение вычетов в разных организациях;
• в рамках одной организации зарегистрировать сотруднику применение вычетов только по одному из мест его работы в этой организации.

Выбор месяца налогового периода

Одним из ключевых сведений о вычетах является месяц налогового периода, начиная с которого эти вычеты будут учитываться при исчислении ИПН.
При создании нового документа месяц заполняется автоматически в соответствии с текущей датой, установленной на компьютере. При необходимости он может быть изменен.

Ввод сведений о предоставляемых вычетах

В документе указывается возможность предоставления следующих вычетов:

  • Вычет ОПВ;
  • Добровольные пенсионные взносы;
  • Расходы на оплату вознаграждений по ипотечным жилищным займам;
  • Расходы на оплату медицинских услуг;
  • Стандартный 1 МЗП;
  • Стандартный 75 МЗП (инвалидам, участникам ВОВ);
  • Стандартный 75 МЗП (родителям инвалидов, приемным родителям).
