Заявление на предоставление вычетов ИПН

Документ предназначен для первоначального ввода и последующего изменения сведений о налоговых вычетах, которые будут учитываться при расчете ИПН. Для зарегистрированных сумм прочих вычетов до 2018 года используется форма документа (до 2018 года). Для регистрации сведений о предоставляемых вычетах с 2018 года используется новая форма документа. В новой форме документа не указываются сведения о льготах, о стандартных и прочих вычетах, а заполняются сведения по видам вычетов, предусмотренных законодательством РК.
Прекращение действия отдельных вычетов также регистрируется этим документом.

Сведения, внесенные в документ, будут применяться начиная с указанного в документе месяца налогового периода, указанного в поле Месяц.
Другими словами, в документ следует вносить исчерпывающую информацию о вычетах, которые следует применять с определенного месяца, все предыдущие сведения (введенные другими документам с меньшим месяцем налогового периода) перестают с этого месяца действовать. В связи с этим недопустимо вводить два заявления с одинаковым месяцем налогового периода.
Прекращение применения вычетов устанавливается посредством заполнения поля Действует по. Если значение данного поля будет не заполнено, то вычеты будут применяться без ограничения срока.

Действие документа и его проведение возможно только при установленном признаке Наличие заявления. Данный признак предназначен для подтверждения получения подписанного сотрудником заявления.

Вычеты, для которых предусмотрен график платежей, применяются согласно данных графика и периода его действия. В графике необходимо указать размер и период действия вычета.

Прекращение применения всех действующих вычетов сотрудника (например, если сотрудник собирается получать вычеты в другой организации) регистрируется документом Прекращение предоставления вычетов ИПН.

Просмотр и изменение всех сведений о вычетах сотрудника (в том числе ввод этих документов) можно выполнять из раздела Налоги карточки сотрудника по гиперссылке Все заявления на вычеты.
