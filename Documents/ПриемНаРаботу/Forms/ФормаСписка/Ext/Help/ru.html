Приемы на работу

Документ предназначен для регистрации факта заключения трудового договора с сотрудником и для оформления соответствующего приказа о приеме на работу.

См. также

  • Для оформления в программе приема на работу всех сотрудников, которые уже работают на предприятии на момент начала использования программы, предназначен специальный документ Данные на начало эксплуатации;
  • Для оформления приема на работу списка сотрудников предназначен документ Прием на работу списком.

Если в программе работают пользователи с ограниченными правами, то не до конца оформленные (не утвержденные) документы в списках выделяются полужирным шрифтом. Утвердить документы можно непосредственно из списка, не открывая их, с помощью кнопки Утвердить.
