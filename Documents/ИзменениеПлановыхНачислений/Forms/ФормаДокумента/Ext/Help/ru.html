Изменение плановых начислений

Документ предназначен для отражения изменения условий оплаты труда (плановых начислений) нескольких сотрудников, в том числе временного.

Его ввод предполагается, например, для всех сотрудников, занимающих определенную позицию штатного расписания, при изменении начислений для этой позиции. В этом случае документ можно ввести непосредственно из карточки позиции или документа, которым изменяются начисления позиции (если ведется история изменения штатного расписания).

Заполнение документа

Таблица сотрудников заполняется автоматически всеми сотрудникам выбранного подразделения и/или выбранной должности (позиции штатного расписания).
Также сотрудников можно подобрать в таблицу непосредственно. По кнопке Заполнить можно перезаполнить список сотрудников сотрудниками этого подразделения и/или этой должности.

По умолчанию для заполнения предлагаются сотрудники, работающие на указанную в шапке дату изменения. Эта же дата (а также дата окончания периода) будет автоматически заполнена в таблицу. Однако даты в таблице можно изменить – именно они будут учтены при назначении начислений сотрудникам.

Если требуется изменить лишь некоторые начисления сотрудников, или назначить/прекратить определенные начисления, то их можно перечислить в таблице по ссылке Отбор начислений не установлен. Если эту таблицу оставить незаполненной, то в документ будут помещены все действующие плановые начисления сотрудников. Установка отбора при уже заполненном документе не влечет его перезаполнения – при необходимости документ следует перезаполнить.

Непосредственно в таблице сотрудников для просмотра и изменения доступны показатели их плановых начислений. Поскольку разные сотрудники могут иметь разные плановые начисления (с разными показателями), то ячейки с неактуальными для сотрудника показателями (но актуальными для других сотрудников) для изменения недоступны.

Для того, чтобы не заполнять в каждой строке таблицы одинаковые показатели или суммы, можно воспользоваться кнопкой Заполнить показатели. В открывшейся при этом форме следует указать данные, которые будут автоматически заполнены в каждой строке таблицы.
