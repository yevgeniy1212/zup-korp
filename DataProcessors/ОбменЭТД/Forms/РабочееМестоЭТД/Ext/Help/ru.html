Рабочее место ЭТД

Рабочее место ЭТД предназначено для работы с электронными трудовыми договорами и дополнительными соглашениями к электронным трудовым договорам.

Рабочее место ЭТД состоит из двух вкладок:

Вкладка "Договоры" состоит из двух разделов:

  • Сведения об основных договорах

Информация в раздел выводится на основании зарегистрированных в системе документов Прием на работу, Прием на работу списком, Начальная штатная расстановка.

  • Сведения о дополнительных соглашениях

Информация в раздел выводится на основании документа Кадровый перевод.

Вкладка "Действия с ЭТД" состоит из трёх разделов:

  • Сотрудники к расторжению

В данном разделе отображаются уволенные сотрудники, электронный трудовой договор которых не был расторгнут. Информация в раздел выводится на основании документов Увольнение, Увольнение списком.

  • Сотрудники к пролонгации

В данном разделе отображаются сотрудники с истекшей датой окончания трудового договора или с истекшей датой пролонгации.

  • Сотрудники к восстановлению

В данном разделе отображаются сотрудники, которые имеют расторгнутый электронный трудовой договор, но по которым не был проведен документ Увольнение, а также ранее уволенные сотрудники, с расторгнутым электронным трудовым договором, по которым был проведен документ Восстановление в должности.

Список кадровых изменений для формирования дополнительных соглашений и список сотрудников к расторжению ЭТД формируются только по зарегистрированным в ЕСУТД договорам (отправленным в ЕСУТД договорам).
Если договор ЭТД сотрудника не зарегистрирован в ЕСУТД (не отправлен) данные по кадровым изменениям такого сотрудника не отражается в рабочем месте ЭТД.

Разделы сгруппированы по действиям, которые необходимо произвести с ЭТД:
  • по ссылке Все выводится список всех электронных трудовых договоров, как уже созданных, так и тех которые необходимо создать;
  • по ссылке Не созданы отображается список сотрудников, по которым требуется создать ЭТД;
  • по ссылке Не отправлены отображается список ЭТД, которые были созданы, но еще не отправлены в ЕСУТД. Для возможности отправить электронный трудовой договор из рабочего места предусмотрена кнопка Отправить в ЕСУТД;
  • по ссылке В ожидании ответа отображается список ЭТД, которые были отправлены, но не обработаны в ЕСУТД. Для обновления статуса подрбных договоров предусмотрена кнопка Обновить статус из ЕСУТД.
