Формирование заявлений на вычеты по ИПН

Обработка предназначена для создания документов Заявление на предоставление вычетов ИПН с 01.01.2026г.

В обработке предоставлена возможность создания заявлений по определенной организации и по выбранным сотрудникам. Есть 2 варианта создания документов: 

- Формирование новых заявлений 

При выборе данного варианта в ходе обработки становится возможным сформировать новые документы "Заявление на предоставление вычетов ИПН" для сотрудников (физических лиц-резидентов), у которых в учёте ещё не назначены вычеты.

- Замена назначенных вычетов

При выборе данного варианта обработка позволяет сформировать новые документы "Заявление на предоставление вычетов ИПН" для сотрудников (физических лиц-резидентов), у которых ранее были назначены вычеты. В этом варианте возможно прекратить действие устаревших вычетов и/или заменить их на новые в соответствии с установленными правилами.

Можно ограничить срок действия вычетов на конец года с помощью настройки "Ограничивать ввод заявлений концом года". 

Настройка "Прекратить действие вычетов, назначенных ранее, но не действующих", автоматически отключает предоставления вычетов, у которых включена опция в справочнике вычетов "Вид вычета больше не используется", при формировании списка физических лиц.

Предоставленные вычеты физическому лицу отображаются в поле Вычеты сотрудника. Для просмотра предоставленных вычетов сотруднику, требуется выделить его в списке. Вычеты, по которым применяется График, в табличной части указывается гиперссылка Подробнее, при нажатии на которую открывается дополнительная форма для просмотра графика платежей; по остальным вычетам указывается только вид вычета.

В случае наличия ранее введенных документов Заявление на предоставление вычетов ИПН система автоматически сопоставляет статусы вычетов ранее действующих и введенных в документе.
