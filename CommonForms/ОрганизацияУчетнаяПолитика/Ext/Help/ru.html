Налогообложение

Настройки особенностей налогообложения организации.

Является вкладчиком ОППВ - данный признак необходимо установить, если организация является вкладчиком обязательных профессиональных пенсионных взносов в пользу работников, занятых на работах с вредными (особо вредными) условиями труда.
Применять ограничение на минимальный объект ОСМС - данный признак необходимо установить, если при расчете отчислений ОСМС требуется применять минимальное ограничение к объекту обложения в размере 1 МЗП.
Предоставлять разрешенный вычет в месяце отсутствия дохода - признак необходимо установить, если разрешенный вычет будет предоставляться в месяце отсутствия дохода. Действует с 01.01.2018 г.
Отражать в НУ удержание налогов и взносов пропорционально выплаченному доходу - признак необходимо установить, если суммы налогов и взносов, подлежащие перечислению, будут рассчитываться от выплаченных доходов работника. При установленном флажке суммы налогов и взносов, подлежащие перечислению, будут рассчитываться с помощью документа Удержание налогов и взносов по налоговому учету.
Применять вычет ОПВ для ГПХ (действует до 2020 г.)
 Действует с 01.01.2019 по 31.12.2019. Законом Республики Казахстан № 382-VI "О внесении изменений и дополнений в Кодекс Республики Казахстан "О налогах и других обязательных платежах в бюджет" (Налоговый кодекс) предусмотрено предоставление физическим лицам по договорам ГПХ вычета по ОПВ при расчете ИПН.
Если признак установлен, при расчете ИПН предоставляется вычет в размере ОПВ.
Если признак не установлен, вычет по ОПВ не предоставляется.
Расчет СН, ООСМС, ВОСМС без применения 90 % корректировки по ИПН (действует в 2020 г.)
Действует с 01.01.2020 по 31.12.2020. Законом Республики Казахстан № 382-VI "О внесении изменений и дополнений в Кодекс Республики Казахстан "О налогах и других обязательных платежах в бюджет" (Налоговый кодекс) предусмотрен расчет СН, ООСМС, ВОСМС без применения 90% корректировки по ИПН.
Если признак установлен, то при расчете социального налога, отчислений и взносов на обязательное социальное медицинское страхование не применяется 90% корректировка облагаемого дохода работников в соответствии с пп. 49) пункта 1 статьи 341 НК РК.
Если признак снят, то корректировка участвует в расчете социального налога и взносов/отчислений по ОСМС.
Применять вычет ВОСМС для физических лиц (сотрудников) (действует в 2020 г.)
Действует с 01.01.2020 по 31.12.2020. Законом Республики Казахстан № 382-VI "О внесении изменений и дополнений в Кодекс Республики Казахстан "О налогах и других обязательных платежах в бюджет" (Налоговый кодекс) предусмотрено предоставление физическим лицам вычета по ИПН:
- по доходам работников вычета на сумму ВОСМС;
Если признак установлен, то при расчете ИПН физическим лицам предоставляются вычеты.
Если признак снят, то вычеты не предоставляются.
Применять вычеты ОПВ и ВОСМС для ГПХ (действует в 2020 г.)
Действует с 01.01.2020 по 31.12.2020. Законом Республики Казахстан № 382-VI "О внесении изменений и дополнений в Кодекс Республики Казахстан "О налогах и других обязательных платежах в бюджет" (Налоговый кодекс) предусмотрено предоставление физическим лицам вычетов по ИПН:
- по доходам гражданско-правового характера вычетов на сумму ОПВ;
- по доходам гражданско-правового характера вычетов на сумму ВОСМС.
Если признак установлен, то при расчете ИПН физическим лицам, работающим по договору ГПХ, предоставляются вычеты.
Если признак снят, то вычеты не предоставляются.

Расчет зарплаты

Размер доплаты за ночные часы - если в настройках расчета зарплаты выбрана возможность регистрировать и доплачивать за работу в ночное, то по умолчанию процент такой доплаты при ее начислении составит 50.
Подробнее о принципе расчета доплаты см. начисление зарплаты и взносов.

Краткосрочные изменения условий труда

Вести учет рабочего времени по условиям труда - признак необходимо установить, если ведется регистрация рабочего времени по условиям труда сотрудника. При включении данного признака учет ОППВ будет переориентирован на данные по условиям труда, зарегистрированные документом Табель учета рабочего времени.

Расчет среднего заработка

Рассчитывать кадровые изменения в пределах мест работы - настройка позволяет учитывать данные кадровых перестановок при расчете коэффициента индексации для среднего заработка с учетом:

  • Должности/разряда.
  • Должности/разряда и/или подразделения.

Порядок учета рабочего времени - настройка определяет, как будет рассчитываться количество отработанных дней (часов) за предшествующие 12 месяцев при расчете среднего заработка:

  • По фактическим графикам сотрудников.
  • По балансу пятидневной недели.
  • По балансу шестидневной недели.
  • Действует с - дата, начиная с которой будет действовать указанная запись.

Порядок учета рабочего времени (в периоде события для суммированного учета рабочего времени) - настройка позволяет определить значение показателя среднемесячного количества рабочих часов (СрмРЧ) при расчете отпуска и компенсации при увольнении по сотрудникам, работающим по суммированному учету рабочего времени, согласно балансу рабочего времени на соответствующий календарный год:

  • По балансу пятидневной недели.
  • По балансу шестидневной недели.
  • По балансу пятидневной недели (36-часовая).
  • По балансу шестидневной недели (36-часовая).

Кадровый учет

День увольнения является рабочим днем - признак необходимо установить, если в организации день увольнения признается последним рабочим днем сотрудника.
Если день увольнения не является последним рабочим днем, то признак должен быть снят.

Перечисление взносов и отчислений

Содержится информация о сроках перечисления налогов, взносов и отчислений.

  • Вид взноса/отчисления - вид налога, взноса, отчисления;
  • Дата перечисления - последняя допустимая дата перечисления суммы налога, взноса, отчисления. Начиная со следующей после указанной даты, платеж считается просроченным и возникает необходимость уплаты пени за несвоевременное перечисление.
  • Определение срока перечисления - значение, определяющее начиная с какого месяца начинается течение срока:
    • По месяцу налогового периода - срок задается относительно месяца, за который был исчислен соответствующий налог, взнос, отчисление;
    • По месяцу выплаты доходов - срок задается относительно месяца, в котором был выплачен доход, являющийся базой обложения соответствующего налога, взноса, отчисления.
  • Действует с - дата, начиная с которой будет действовать указанная запись.

Округление налогов, взносов и отчислений

Содержится информация о порядке округления налогов, взносов и отчислений при исчислении.

  • Вид налога/взноса/отчисления - вид налога, взноса, отчисления;
  • Способ округления - из справочника Способы округления зарплаты выбирается требуемый способ округления при исчислении налогов, взносов и отчислений.
  • Действует с - дата, начиная с которой будет действовать указанная запись.
