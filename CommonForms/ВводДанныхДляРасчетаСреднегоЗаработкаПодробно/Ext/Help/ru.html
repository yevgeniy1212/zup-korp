Ввод данных для расчета среднего заработка подробно

При расчете среднего заработка для отпусков, больничных и т.п. в таблице представлена расшифровка суммы, учитываемой в расчете среднего заработка:

  • для всех видов премий – количество месяцев расчетного периода;
  • для полностью учитываемых премий – дата выплаты премии;
  • для годовых премий – год, за который начислена премия.

Для сумм, которые при расчете среднего заполнены автоматически согласно ранее зарегистрированным в программе начислениям, сведения доступны только для просмотра.
Если суммы вводились при расчете среднего заработка вручную, то сведения по ним здесь допускается уточнять.
